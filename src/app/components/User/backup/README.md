# Backup Landing Page Components

This directory contains the original landing page components that were replaced with a new design.

## Components

- **HeroBanner.jsx** - Original hero banner with Skill-Learn logo and tagline
- **Features.jsx** - Original features section with feature cards
- **HowItWorks.js** - Original "How It Works" section with steps
- **Testimonials.jsx** - Original testimonials section
- **page.jsx** - Original landing page implementation

## Usage

These components can be restored if needed by:
1. Moving them back to `src/app/components/User/`
2. Updating `src/app/page.jsx` to import from the correct paths
3. Updating the layout if necessary
